# 🌐 Внешнее подключение к серверу

## 🎯 Для владельца сервера (вы):

### 1. Запуск сервера для внешних подключений:
```bash
# Запустить сервер на всех интерфейсах
cd /home/ubuntu/OnlyFamcs/server
source venv/bin/activate
python main.py

# Сервер будет доступен на:
# - Локально: http://localhost:8080
# - Внешне: http://84.233.195.37:8080
```

### 2. Открыть порт 8080:
```bash
# В настройках роутера открыть порт 8080
# Перенаправить порт 8080 на IP вашего компьютера
```

### 3. Настроить клиент для разработки:
```qml
// В SimpleApp.qml установить:
var isDevelopment = true; // Для локальной разработки
```

---

## 🌍 Для внешних пользователей:

### 1. Сборка проекта:
```bash
git clone https://github.com/ВАШ_НИК/video-chat-roulette.git
cd video-chat-roulette

# Установить зависимости
sudo apt update
sudo apt install qt6-base-dev qt6-multimedia-dev qt6-websockets-dev cmake build-essential

# Собрать
./build.sh
```

### 2. Настроить клиент для внешнего подключения:
```qml
// В SimpleApp.qml изменить:
var isDevelopment = false; // Для подключения к внешнему серверу
```

### 3. Запуск:
```bash
./VideoChatRoulette
```

---

## 🔄 Переключение режимов:

### Разработка (localhost):
- `var isDevelopment = true;`
- Подключается к `http://localhost:8080`
- Для локального тестирования

### Продакшн (внешний сервер):
- `var isDevelopment = false;`
- Подключается к `http://84.233.195.37:8080`
- Для внешних пользователей
